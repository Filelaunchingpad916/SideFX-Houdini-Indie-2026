## 💡 Using Houdini Indie

Houdini uses a node-based workflow. This means you connect different tools to create a final result. Each node performs a specific task. You can go back to any previous step to make changes. This freedom helps you iterate on your ideas without discarding past work.

### Your First Project
Start by creating a box in the viewport. Press the Tab key to open the node menu. Type "box" and press Enter. Click in the viewer area to drop the primitive into your scene. Now add a node to modify it, such as a "subdivide" node. Connect the output of the box node to the input of the subdivide node. You will see the change immediately.

### Understanding Assets
You can save your node networks as digital assets. These files act like custom tools. You can move these tools between different projects. This saves time when you work on repeating tasks.

## ⚠️ Troubleshooting Common Issues

Software installation can sometimes hit small snags. Follow these suggestions if you face blockers.

### Slow Viewport
If the 3D scene moves slowly, check your graphics settings. Go to the display options in your viewport. Turn off high-quality lighting or anti-aliasing features. This helps the computer process the movement faster.

### Missing Libraries
If the program fails to launch, verify that your graphics drivers are current. Visit the website of your graphics card manufacturer to find the latest updates for your specific model.

### Permission Errors
If the installer stops during the setup, try running the file as an administrator. Right-click the installer icon and select "Run as administrator" from the menu.

### License Issues
If you cannot verify your license, check your internet connection. Ensure that your firewall does not block the software from reaching the SideFX servers. You can check the status of your account on the official SideFX website.

## 📁 Managing Your Files

Keep your work organized to avoid losing your data. Houdini creates unique file types. Always save your project before you close the viewport. Use specific names for each version of your file. Houdini handles large amounts of geometry data. Create a dedicated folder for each project. Store your textures, images, and model data within this folder. This keeps the file paths clear.
